# ComfyUI React Frontend

A modern React frontend application for ComfyUI.

## Features

- React with TypeScript
- Vite for fast development
- Modern CSS styling

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS3
